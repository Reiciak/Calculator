# Calculator

A simple calculator application built with Flutter and Riverpod.

![Calculator](docs/Calculator.png)

## Getting Started

This project is a starting point for a Flutter application.

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install)

### Installation

1. Get dependencies:
   ```bash
   flutter pub get
   ```

### Running the Project

You can run the project using standard Flutter commands or the provided `Makefile`.

#### Using Flutter CLI

To run on your default device:
```bash
flutter run
```

#### Using Makefile

- Run on Chrome:
  ```bash
  make run_chrome
  ```
- Run on Linux:
  ```bash
  make run_linux
  ```

## Project Structure

- `lib/`: Contains the source code.
- `docs/`: Contains documentation and images.
